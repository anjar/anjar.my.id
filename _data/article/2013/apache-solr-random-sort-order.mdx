Random Sort Order Apache SOLR adalah salah satu cara untuk menghasilkan result random di apache solr seperti halnya "*order by rand()*" di MySQL adalah dengan menambahkan dynamic field random di schema.xml.
Pertama tambahkan type field random berikut di *schema.xml* 

```xml
<fieldType  name="rand"  class="solr.RandomSortField"  indexed="true" />
```
Kemudian tambahkan field random di schema.xml 

```xml
<dynamicField  name="random*"  type="rand"  indexed="true"  stored="false" />
```
Kemudian restart solr.&nbsp;Untuk mendapatkan hasil random dapat dengan mengorder by random field , contoh sort=random_1234 , buka browser lalu coba untuk random lalu akses di browser.
